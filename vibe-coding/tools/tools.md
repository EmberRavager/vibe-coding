# AI 编程工具下载与优劣

这份清单只保留最有用的工具。你不需要全装，选一个适合的就够用。

## 工具清单

- Cursor
- Claude（Claude Code / Claude Web）
- Trae
- Codex（Codex CLI）
- OpenCode

## 下载、价格与优劣对比

### Cursor

- 下载：https://cursor.com/
- 价格：以官网订阅页面为准
- 优点：IDE 级 AI 体验强，适合中型项目协作
- 缺点：对规则与上下文配置要求更高

### Claude（Claude Code / Claude Web）

- 下载：
  - Claude Web：https://claude.ai/
  - Claude Code：https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/overview
- 价格：以官网订阅页面为准
- 优点：长上下文与复杂任务表现好
- 缺点：需要配置规则与权限

### Trae

- 下载：以官网为准（建议通过官方渠道获取）
- 价格：以官网为准
- 优点：一体化 AI 开发体验，适合端到端任务
- 缺点：不同模型表现差异大，规则配置依赖度高

### Codex（Codex CLI）

- 下载：https://chatgpt.com/codex
- 价格：以官网订阅页面为准
- 优点：终端原生、可脚本化、适合自动化
- 缺点：需要熟悉命令行

### OpenCode

- 下载：https://github.com/opencode-ai/opencode
- 价格：开源免费，模型调用可能产生费用
- 优点：开源可控、可自托管、适合深度定制
- 缺点：需要自己配置模型与环境

## 中国用户使用说明

1. 访问与下载：优先使用各工具官网或官方 GitHub
2. 账号与支付：部分产品需要国际支付方式，价格以官网为准
3. 网络与模型：如访问受限，可考虑官方支持的本地/代理方案或选择开源工具
4. 合规与隐私：生产项目需确认数据上传与合规要求

## 常见问题

**Q：我应该选哪个？**
想快上手就选 Cursor 或 Claude Web，想走终端自动化就选 Codex CLI 或 OpenCode。
