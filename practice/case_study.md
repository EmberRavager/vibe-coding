# 完整实战案例（从需求到验证）

我们用一个真实的产品场景走一遍流程。你看完就能照着做自己的功能。

## 场景

用户需要在“订单列表页”导出 CSV，用于对账。

## 目标与验收

- 目标：在订单列表页新增“导出 CSV”功能
- 验收：点击按钮后能下载 CSV，字段包含订单号、金额、状态、创建时间


## 步骤 3：小步实现

```text
按小步实现，一次只改一个点。
```

建议顺序：

1. 添加按钮与触发逻辑
2. 构建 CSV 数据
3. 触发下载
